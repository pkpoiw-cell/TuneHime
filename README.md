# AI Live Tuner

AI Live Tuner 是一个直播人声实时修音软件原型，目标场景是：

```text
麦克风 -> AI Live Tuner -> 虚拟声卡/监听设备 -> OBS/直播伴侣
```

当前版本定位为商业项目的内测版基础工程，已经具备实时音频链路、参数调节、预设、配置保存、日志和打包脚本。正式上线前仍需要替换更高质量的修音引擎，并完成虚拟声卡、代码签名、安装包和大规模兼容测试。

## 已实现

- 麦克风输入和输出设备选择
- 实时音高检测
- 专业音频引擎：优先使用 pedalboard 的 PitchShift、NoiseGate、Compressor、HighShelf、Limiter
- 根音和调式选择
- 智能识别调号：录制/分析人声音高后自动设置调号和大调/小调
- 一键直播模式：自动识别输入音频并自动调整修音、人声增强和输出音量
- 修音强度、修音速度、干湿比
- 噪声门、压缩、亮度增强、软限幅
- 内置预设：自然、流行、电音
- 自定义预设保存和删除
- 三页中文界面：直播控制、效果测试、虚拟声卡
- 录制 5 秒测试音频，对比原声和处理后效果
- 自动查找虚拟声卡输出，并可发送测试音
- 参数、设备和预设自动保存
- 日志文件，便于售后排查音频设备问题
- 单元测试
- PyInstaller 打包脚本

## 一键直播使用方式

现在默认推荐使用“一键直播”页，不需要手动调参数：

```text
1. 输入选择你的麦克风
2. 输出选择耳机、声卡，或 CABLE Input
3. 勾选“自动识别调号并自动调参”
4. 点击“一键开始直播修音”
5. 正常唱歌即可，软件会边听边自动调整
```

自动模式会实时做这些事：

```text
自动识别更可能的调号和大调/小调
自动调整修音强度和修音速度
自动调整噪声门，减少环境底噪
自动调整压缩，让音量更稳定
自动调整亮度，让人声更靠前但不过刺
自动调整输出增益，降低忽大忽小
```

高级设置页只用于调试和特殊场景。普通用户不用打开。

## 手动使用方式

### 1. 选择音频设备

打开软件后，先在“音频设备”里选择：

```text
输入：你的麦克风
输出：你的耳机、声卡，或虚拟声卡的输入端
```

如果只是自己试听，输出可以先选耳机或扬声器。

如果要接直播软件，推荐安装 VB-CABLE 后这样选：

```text
AI 直播修音的“输出”选择 CABLE Input
OBS/直播伴侣的“麦克风”选择 CABLE Output
```

### 2. 选择调号和调式

在“修音”区域选择：

```text
调号：歌曲的主调，比如 C、D、A#
调式：大调、小调或半音阶
```

不知道歌曲是什么调时，可以先选“半音阶”，它不会限制到某个大调/小调内，修起来更宽松。

也可以点击“智能识别调号”。软件会录 5 秒人声，分析音高分布，然后自动设置调号和调式。

### 3. 选择预设

先从“预设”里选一个：

```text
自然：轻修音，适合聊天、自然唱
流行：更稳定，适合大多数直播唱歌
电音：强修音，适合明显电音效果
```

选中后点“应用”。

### 4. 开始监听

点击“开始”。正常情况下：

```text
检测音高会显示当前人声音高
目标音会显示修正到哪个音
修正量会显示当前拉了多少音高
```

点击“停止”可以停止音频处理。

### 5. 常用调参

```text
修音强度：越高越准，但越容易有电音感
修音速度：越高越快，越低越自然
干湿比：修音后声音占比，越高修音越明显
噪声门：环境噪声大时调高，吞字时调低
压缩：音量忽大忽小时调高
亮度：声音不够靠前时调高，刺耳时调低
增益：整体输出音量
旁路修音：临时关闭修音，保留基础输出
```

### 6. 保存自己的预设

调好参数后：

```text
点击“保存”
输入预设名称
以后可在预设下拉框中直接选择
```

## 测试自己的效果

打开“效果测试”页：

```text
1. 点击“录制 5 秒”
2. 对着麦克风唱或说一段
3. 点击“播放原声”
4. 点击“播放处理后”
5. 点击“A/B 连续对比”可以先听原声，再听处理后
```

这个功能不会进入直播软件，只用于你自己听效果。调参时推荐反复用它测试。

## 虚拟声卡模式

打开“虚拟声卡”页：

```text
1. 先安装 VB-CABLE 或 Voicemeeter
2. 点击“刷新设备”
3. 点击“自动查找虚拟声卡”
4. 如果找到 CABLE Input，软件会自动选中它作为输出
5. 点击“发送 1 秒测试音”
6. 在 OBS/直播伴侣里看麦克风音量条是否跳动
```

直播软件中这样设置：

```text
AI 直播修音 输出：CABLE Input
OBS/直播伴侣 麦克风：CABLE Output
```

注意：当前软件提供的是“虚拟声卡路由效果”，即把处理后的声音送到已安装的虚拟声卡。真正内置一个系统级虚拟麦克风需要单独开发和签名音频驱动。

## 直播软件接入

当前版本可以输出到任意系统播放设备。要接 OBS、抖音直播伴侣、B 站直播姬等软件，建议先用 VB-CABLE 验证：

```text
AI Live Tuner Output 选择 CABLE Input
直播软件麦克风选择 CABLE Output
```

正式商业版建议做自有虚拟麦克风驱动或 OBS/VST3 插件，降低用户配置成本。

## 配置和日志

Windows 默认路径：

```text
%APPDATA%\AI Live Tuner\settings.json
%APPDATA%\AI Live Tuner\logs\app.log
```

如果该目录不可写，程序会退到系统临时目录。也可以通过环境变量指定数据目录：

```bat
set AI_LIVE_TUNER_DATA=D:\AI Live Tuner Data
```

## 当前限制

当前已经优先使用 pedalboard 的专业音频处理模块；如果运行环境缺少 pedalboard，会自动退回基础引擎。商业正式版仍建议继续做大量真实人声素材测试，并根据测试结果调整参数或替换为更低延迟的专用 C++ 引擎。

另一个重要限制是虚拟麦克风。目前推荐用户安装 VB-CABLE。商业正式版需要减少这个门槛。
