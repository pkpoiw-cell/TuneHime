## 核心组件设计

### 1. 音高仪表盘（半圆形 + 可爱元素）

```
        ╭─────────────────────╮
       ╱   🎵 偏高            ╲
      ╱  ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄  ╲
     │  🔴    🟡    🟢    🟡    🔴 │
     │         ╲  │  ╱           │
     │          ╲ │ ╱            │
     │           ╲│╱             │
     │            ▼              │
      ╲                         ╱
       ╲      ♪ A4             ╱
        ╲    440 Hz           ╱
         ╰─────────────────╯
            +12 cents
```

- 圆弧用渐变色：红→黄→绿→黄→红
- 指针带动画，有弹性回弹效果
- 中间显示音名和频率
- 偏差大时指针抖动，回到中心时有"叮"的动画

### 2. 电平表（竖条 + 峰值）

```
  ┌──┐
  │▓▓│ ← 绿色 (0-60%)
  │▓▓│
  │▓▓│
  │▓▓│ ← 黄色 (60-85%)
  │▓▓│
  │░░│ ← 红色 (85-100%)
  │░░│
  │──│ ← 峰值线
  │░░│
  └──┘
  -12  -6   0   +3 dB
```

### 3. 频谱可视化（波浪形）

```
  ╭─────────────────────────────────╮
  │  ▁▂▃▅▇█▇▅▃▂▁▂▃▅▇▅▃▂▁▂▃▅▃▂▁  │
  │  ▁▂▃▅▇█▇▅▃▂▁▂▃▅▇▅▃▂▁▂▃▅▃▂▁  │
  ╰─────────────────────────────────╯
    80Hz                      8000Hz
```

- 用渐变色：低频紫→中频粉→高频白
- 带呼吸感的脉动动画

### 4. 按钮样式

```
  主按钮（开始修音）:
  ╭──────────────────────╮
  │   ♪ 一键开始直播修音   │  ← 樱花粉渐变，hover 发光
  ╰──────────────────────╯

  次要按钮（停止）:
  ╭──────────────╮
  │     停止      │  ← 半透明紫，hover 变亮
  ╰──────────────╯

  滑块:
  噪声门 ────────●──────── 0.35
                 ↑ 带圆形把手，拖动时有光晕
```

### 5. Tab 标签页

```
  ╭──────╮╭──────╮╭──────╮╭──────╮
  │🎤直播││⚙设置 ││🎧测试││🔌声卡│
  ╰──────╯╰──────╯╰──────╯╰──────╯
     ↑ 选中状态：粉色下划线 + 发光
```

### 6. 可爱元素点缀

- 标题旁的音符图标（♪♫♬）
- 启动时有简短的加载动画（跳动的音符）
- 检测到音高时，音名旁有小星星闪烁
- 修音启动时，状态栏有"✨修音中..."的动画
- 错误提示用可爱的语气："哎呀，麦克风没找到呢～"

## 动画设计

| 触发 | 动画 |
|------|------|
| 启动 | Logo 渐入 + 音符飘落 |
| 开始修音 | 按钮脉冲发光 + 状态渐变 |
| 音高变化 | 指针弹性回弹（spring animation） |
| 切换 Tab | 内容滑入（200ms ease-out） |
| 检测到调号 | 星星闪烁 + 文字弹入 |
| 错误 | 卡片轻微抖动（shake） |

### 架构

```
┌─────────────────┐    WebSocket    ┌─────────────────┐
│   Electron UI   │ ◄────────────► │  Python Backend  │
│   (HTML/CSS/JS) │    JSON 消息    │  (audio_engine)  │
│                 │                 │                  │
│  • 仪表盘       │  ◄── 音高数据   │  • 麦克风输入    │
│  • 频谱         │  ◄── 频谱数据   │  • 音频处理      │
│  • 滑块         │  ──► 参数变更   │  • 输出到声卡    │
│  • 按钮         │  ──► 开始/停止  │                  │
└─────────────────┘                 └─────────────────┘
```

Python 后端通过 WebSocket 实时推送音高、频谱、电平数据到前端，前端只负责渲染。
前端的参数变更（滑块、按钮）通过 WebSocket 发送给后端。
